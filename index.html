Vidada is a free video and image manager which supports custom Tags and supports nearly every video format. You can have thousands of videos and images indexed, and all can be visualized in the thumbnail view. Join the Vidada expirience!

Supporting all formats

Vidada is backed up with ffmpeg and VLC, which makes it work with almost all known video formats.

Powerful Tags

Tags are a very powerful way to index your media library and to search in it. Vidada provides several comfort functions to make it easy to use.

AutoTag

You have a nice folderstructure and describtive file names? With AutoTag, matching Tags are applyed for you automatically.

DirectPlay

Instantly play any video in your Thumbnail Viewer. Move your mouse over the playing viedo and it will seek to the relative position. Figuring out whats inside a video was never simpler.

Platfrom independed

Vidada works on any platform. You can even sync your media library between diffrent platforms.

Does not touch your filestructure

Vidada does not touch your file structure and also does not depend on it. You can work with your media files the way you want.

HDPI / Retina ready

Vidada supports HDPI rendering, this way everything looks crisp sharp on your Retina Display.

Media Libaray

This is the basic Thumbnail Viewer, above you can see the filter in action. At the bottom, there is the media detail for the current selected item. You can also select multiple items and edit them at once.

Tag Management

In the Tag manager, you can create new Tags and edit existing ones. You can specify the AutoTag behaviour and the keywords which AutoTag will use.

Zoomed Thumbs

You can zoom in and out the preview thumbs easily. The design is responsive, the smaller the Thumbs are, the less information is displayed on them.

The Vidada project has been started, as there was no existing software which could satisfy these core requirements:

  • A nice zoomable Thumbnail-View for videos
  • Tag the media items with custom tags
  • Run on Windows and OS X
  • Support the Retina Display

While in development, much more, unquie features have been implemented. The DirectPlay feature, which lets you play a movie right away in the Thumbnail Viewer is another outstanding feature among many others.

Vidada gives you a lot of comfort viewing and finding your medias without forcing you to do anything. The AutoTag feature is a great example for this: Vidada can Tag your medias soley based upon your file structure. So if you have already put some effort into your file structure Vidada can use is and do the Tagging for you.

And yes, Vidada is completly free.
